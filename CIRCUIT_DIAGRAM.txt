┌─────────────────────────────────────────────────────────────────────────────┐
│                    MONTAGEM EM PROTOBOARD (VISTA SUPERIOR)                   │
└─────────────────────────────────────────────────────────────────────────────┘

    Coluna →  a b c d e   f g h i j
            ┌─────────────────────────┐
    Linha 1 │ • • • • •   • • • • • │  ←─── Trilha positiva (+)
            │─────────────────────────│
    Linha 2 │ • • • • •   • • • • • │
    Linha 3 │ • • • • •   • • • • • │
    Linha 4 │ • • ⓥ • •   • • • • • │  ← DHT22 Pino VCC
    Linha 5 │ • • ⓓ • •   • ⓡ • • • │  ← DHT22 Pino DATA + Resistor
    Linha 6 │ • • ⓧ • •   • • • • • │  ← DHT22 Pino NC (não conectado)
    Linha 7 │ • • ⓖ • •   • • • • • │  ← DHT22 Pino GND
    Linha 8 │ • • • • •   • • • • • │
            │─────────────────────────│
    Linha 9 │ • • • • •   • • • • • │  ←─── Trilha negativa (-)
            └─────────────────────────┘

    Legenda:
    ⓥ = VCC do DHT22      (vermelho)
    ⓓ = DATA do DHT22     (amarelo/verde)
    ⓧ = NC não conectado
    ⓖ = GND do DHT22      (preto)
    ⓡ = Resistor 10kΩ

    Conexões:
    • Linha 4 (VCC DHT22) → Trilha + → ESP32 3.3V (cabo vermelho)
    • Linha 5 (DATA DHT22) → ESP32 GPIO4 (cabo amarelo/verde)
    • Resistor 10kΩ entre Linha 4 e Linha 5
    • Linha 7 (GND DHT22) → Trilha - → ESP32 GND (cabo preto)

┌─────────────────────────────────────────────────────────────────────────────┐
│                        INFORMAÇÕES IMPORTANTES                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  ⚠️  CUIDADOS:                                                               │
│                                                                              │
│  1. Use APENAS 3.3V do ESP32, NÃO use 5V (pode danificar o ESP32)         │
│  2. Verifique a polaridade antes de ligar                                  │
│  3. O resistor pull-up é OBRIGATÓRIO para comunicação estável             │
│  4. Alguns módulos DHT22 já incluem o resistor integrado                   │
│  5. Mantenha cabos curtos (menos de 30cm) para evitar interferências      │
│                                                                              │
│  📝 NOTAS:                                                                   │
│                                                                              │
│  • O DHT22 pode levar 2-3 segundos para estabilizar após ligar            │
│  • Intervalo mínimo entre leituras: 2 segundos                             │
│  • Precisão de temperatura: ±0.5°C                                         │
│  • Precisão de umidade: ±2-5%                                              │
│  • Faixa de temperatura: -40°C a 80°C                                      │
│  • Faixa de umidade: 0% a 100% RH                                          │
│                                                                              │
│  🔧 ALTERNATIVAS DE PINO GPIO:                                              │
│                                                                              │
│  Se GPIO4 estiver ocupado, pode usar outros pinos digitais:                │
│  • GPIO5, GPIO12, GPIO13, GPIO14, GPIO15, GPIO16, GPIO17, etc.            │
│  • Evite: GPIO0, GPIO2, GPIO12 (usados no boot)                           │
│  • Lembre de alterar no código: #define DHTPIN XX                         │
│                                                                              │

│  ☐ Sem curtos-circuitos visíveis                                            │
│  ☐ Cabos bem conectados e firmes                                            │
│  ☐ Polaridade verificada (VCC não conectado ao GND)                         │

┌─────────────────────────────────────────────────────────────────────────────┐
│                          TESTE DE CONTINUIDADE                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│  Com multímetro no modo de continuidade/resistência:                        │
│                                                                              │
│  1. ESP32 3.3V ←→ DHT22 VCC: deve ter continuidade (bip)                   │
│  2. ESP32 GPIO4 ←→ DHT22 DATA: deve ter continuidade (bip)                 │
│  3. ESP32 GND ←→ DHT22 GND: deve ter continuidade (bip)                    │
│  4. VCC ←→ GND: NÃO deve ter continuidade (sem bip)                        │
│  5. Entre VCC e DATA: deve medir ~10kΩ (resistor)                          │
